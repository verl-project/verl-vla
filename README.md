# verl-vla

A unified VLA post-training framework for human-in-the-loop data collection, fine-tuning, and reinforcement learning.
Built on top of [verl](https://github.com/verl-project/verl).

## Project Status

verl-vla is under active development. Its interfaces, configurations, and workflows may change as the project
evolves.

## Documentation

The latest documentation is available at [verl-vla.readthedocs.io](https://verl-vla.readthedocs.io/en/latest/).
